# vul-plugin-webhook

## Installation
```shell
vul plugin install github.com/aquasecurity/vul-plugin-webhook
```

## Usage
```shell
vul webhook -- <plugin flags> -- <vul args>
```

## Command Line Flags

| Flag  | Description | Required | Example               |
|-------|-------------|----------|-----------------------|
| --url | Webhook URL | Yes      | `--url="example.com"` |
